# realcountrysize

一个可交互的国家或地区面积对比网页。

它用球面面积计算来避免常见世界地图投影带来的面积视觉误差，并通过可旋转地球仪的形式，让你把一个国家或地区拖到另一个国家或地区上方，做更直观的大小比较。

## 功能

- 支持点击国家或地区查看面积、排名和面积比例
- 支持拖拽任意国家或地区，生成可叠加的对比副本
- 支持地球仪在拖拽过程中自动平滑旋转，把比较区域带回中央视野
- 使用球面几何面积计算，而不是平面投影面积
- 将台湾并入中国一起渲染

## 技术实现

- `index.html`: 页面结构与信息面板
- `styles.css`: 页面视觉样式、地球仪外观和响应式布局
- `app.js`: 地图加载、国家或地区计算、拖拽交互、地球仪旋转逻辑

项目是一个零构建步骤的静态前端页面，没有使用打包器，直接启动本地静态服务即可运行。

## 本地运行

在项目目录执行：

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://127.0.0.1:4173/
```

## 数据来源

- 世界地图数据：`world.geojson`
- 运行时通过公开 GeoJSON 地址加载
- 渲染与交互基于 [D3.js](https://d3js.org/)

## 使用方式

1. 打开页面后，点击任意国家或地区查看基础信息。
2. 拖动某个国家或地区，会生成一个半透明副本。
3. 将副本移动到另一个国家或地区上方，查看两者面积倍数关系。
4. 点击“清除叠加副本”或双击地图空白处，可以清空对比副本。
